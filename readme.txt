项目名称：meixi（美西网）；(开发目录)
	  xiangmu（发布目录）

项目简介：本网站属于奢侈品时尚网购平台，提供国际一线奢侈品品牌包包,服装,配件,鞋履等系列产品；

模块介绍：有七大模块：
	首页(index)，
	注册页(register)，
	登录页(login)，
	商品列表页(goodslist)，
	商品详情页(goodsdetail)，
	购物车页(shopping)，
	支付页(payment)

；
5.使用流程：   首页：（1）顶部：登录，注册，购物车等快捷登录，品牌logo，导航栏（均可点击直接跳转至相应页面）；
		     （2）中部：展示近期热销产品：轮播图，特别推荐，男仕，女士，正被选购四大模块；
		     （3）底部：网站周边，已经网站验证；

	  注册，登录：均使用后台数据，以及正则验证；（注册，登录成功均可跳转至首页）；

	  商品列表页：列表信息，使用后端数据动态产生；（点击可跳转至商品详情页）；

	  商品详情页：详情信息，使用后端数据动态产生；使用了放大镜功能；（点击可跳转至购物车页）；

	  购物车页面：使用了全选功能，购物车点击功能；（点击可跳转至支付页面）。
